Way of work

Passion for Design

1. Preparing
2. Design
3. Development
4. Delivery
1. Preparing

In this part, I and my clients discuss about the website and take keynotes on what the client needs. It’s very important because when I will make the design of the website I will focus on the keynotes will help me to focus on what I need to do. 

2. Design

After the discussion, I will start designing the website. First, I will make a structure of the website and approve the website structure with the client then I will start putting things together. 

3. Development

If my client is happy with the design I will start developing the main website. Installing Important Plugins and themes for the project. As I’m working the client can view the progress with a link that I will provide. It will help the client to give his feedback in time.

4. Delivery

Alter finishing the project I will give a short preview of all the pages and design (if there any) and all the requirements client requested. Then I will put the website in live so that anyone can see the website.
